Extra functions

This section was created so the user can add a certain openCV function not available in the plataform.

How this works?

The user has to upload a file containing the code of the desired function. Also, he has to introduce the number of arguments and a name to the function.

In the file given there is a certain pattern. The first argument must be the input image and the second the output image. The following arguments are the arguments of the function depending on, how many it requires.
