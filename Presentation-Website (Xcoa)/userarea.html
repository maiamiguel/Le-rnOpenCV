How to use it

User Area

HOW TO USE THE PLATFORM?

Login and Registration

After entering the welcome page the user is greeted by a login page where he can enter his credentials to login to his account.

For the first time user a registration link is available so that he can navigate to the registration form.

  • OpenCV function panel

    On the left of the application the user can access the OpenCV function, through a panel of organized labels. Each category is divided into several functions that are based on a different algorithm.

    Each function has an info button that, when pressed will display information about it.

  • Upload zone

    The upload zone allow users to choose and upload images from their devices. The upload can be done by simple drag & drop or through the navigation in the user's file system.

    It is also used as the display location for all the transformations made.

  • Histogram and Script

    When analyzing a picture it is very handy to have a histogram to show the color distribution in the picture. A click on it presents a bigger image of the histogram, that provides a better understanding of the transformations.

    The script regions shows the sequence of the transformations that were applied to the image. This allows a better understanding of the transformations made.

  • Terminal interface

    Through the terminal interface the user can access and visualize the source code used in the algorithm that provided a certain result. It is also a another way to interact with the application.

    The data shown brings a new layer of information to the learning process.

Applying effects and testing algorithms

The image that are being produced will be shown in the image gallery below the main image.

Every picture has a star that allows the user to save it to his favorite images. This way he can store information about the image, but also about the algorithm used.

Favorite images

Images saved will be available for later sessions. The corresponding information can be viewed on the favorites page.

Not only can the user see and download the image, it is also possible for him to see and download the script associated with the transformations applied to the image.

Custom Scripts

The sequence of effects applied to a picture is presented to the user through a script in pseudo language.

To improve the user experience the script is available for download and to create a custom function. This way the users will create a custom section with a function that applies the sequence of effects.

Add new functions

The number of functions present in the OpenCV libary is huge, which makes it hard to offer all to the user.

To provide expandability to the project the user can bring the source code of OpenCV functions and add them to the personal library. The function name, the number of input arguments and the source code file can be uploaded and will be associated with the user's account.
